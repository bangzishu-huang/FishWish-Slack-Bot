# FishWish-Slack-Bot
A slack bot that will share to you cats images, gifs, and facts!

I wanted to create a slack bot that was dedicated to cats because I had a cat myself, so I thought it would be very cool! This was a really nice project for me to enhance my skills in JavaScript, while also learning how to host a slack bot that will run on its own 24/7.

# Features
FishWish does feature a couple of commands including:
- `/fishwish-ping` - This tells you the current ping of FishWish
- `fishwish-catfact` - A random cat fact for you to know
- `/fishwish-catgif` - Any type of cat gif beyond your wildest imaginations
- `/fishwish-catpic` - Funny cat, serious cat? You never know what cat picture you'll get
- `/fishwish-catfilter` - Cat pictures with random filters thrown onto it
- `/fishwish-greet` - Get greeted be a cat
- `/fishwish-help` - The overview for all the commands available
